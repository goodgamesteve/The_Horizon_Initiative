# Project Horizon Initiative

Ordinary is overrated.

Project Horizon is an open educational initiative dedicated to transforming technology education into immersive engineering experiences that inspire curiosity, creativity, and lifelong learning.

Our mission is simple:

Explore.
Build.
Connect.

This repository contains the complete Project Horizon curriculum framework, teacher resources, student materials, innovation center guides, graphics, templates, and supporting resources developed by educators who believe school should be memorable—not memorization.

Welcome to the Horizon.
